RSA中dp泄露的广义解法

dpd_pdp​泄露应该是每个密码手必会的基础攻击之一了，前段时间复习了一下这些基础攻击，发现了一种用Coppersmith的dpd_pdp​泄露解法，这种方法不再需要eee可枚举，所以是一种更广义的解法

后来我把这道题放在Sloth的选拔赛上，结果是全防出去了啊，如果今年招不到密码手的话锅估计是我来背了x

复习·

先来简单地复习一下普通的dpd_pdp​泄露解法，总所周知，dp≡d(modp−1)d_p \equiv d \pmod {p-1}dp​≡d(modp−1)，所以把这个关系代入ed≡1(modφ(n))ed \equiv 1 \pmod {\varphi(n)}ed≡1(modφ(n))中可得：

edp≡1mod  (p−1)e d_p \equiv 1 \mod {(p-1)} edp​≡1mod(p−1)

展开即存在k<ek < ek<e使得：

edp−1=k(p−1)e d_p - 1 = k (p-1) edp​−1=k(p−1)

即：

p=edp−1k+1p = \frac{e d_p - 1}{k} + 1 p=kedp​−1​+1

一般情况下，dpd_pdp​泄露的题目都会给一个可以枚举的eee，通常是e=65537e=65537e=65537，由于k<ek<ek<e，所以可以通过枚举kkk算得ppp，然后对nnn进行分解

但在这个题目中e=2⋅1076−3e = 2 \cdot 10^{76}-3e=2⋅1076−3，显然不可枚举，于是不能用这种解法

思路·

稍微分析一下可知nnn是2048 bits，所以eee相对于nnn是“小”的，于是可以首先想到用格去解，或者用Coppersmith

下面构造一下Coppersmith的式子，上面已经计算得：

edp−1=k(p−1)e d_p - 1 = k (p-1) edp​−1=k(p−1)

整理一下即：

edp−1+k=kpe d_p - 1 + k = k p edp​−1+k=kp

模ppp：

edp−1+k≡0(modp)e d_p - 1 + k \equiv 0 \pmod p edp​−1+k≡0(modp)

令A=edp−1A = e d_p - 1A=edp​−1，令x=kx = kx=k，就是解同余方程：

A+x≡0(modp)A + x \equiv 0 \pmod p A+x≡0(modp)

这个方程的解法和RSA的素数ppp部分泄露解法相同，即使用Coppersmith解方程

A+x≡0(modn)A + x \equiv 0 \pmod n A+x≡0(modn)

中未知数xxx的一个解，由于ppp是nnn的因子，所以xxx也是A+x≡0(modp)A + x \equiv 0 \pmod pA+x≡0(modp)的一个解
